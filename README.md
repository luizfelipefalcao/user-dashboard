## Getting Started

```bash
npm install
# or
yarn install
to install the application

and 

npm run start
# or
yarn start
to run the application

```

Open [http://localhost:3003](http://localhost:3003) with your browser to see the result.

## Commands

- `start`: starts the application on `localhost:3003`
